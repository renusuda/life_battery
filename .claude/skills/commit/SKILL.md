# Git Commit

- Conventional commits: `feat:`, `fix:`, `refactor:`, `docs:`, `test:`, `chore:`, `ci:`
- Keep commit messages concise (1-2 sentences)
- If pre-commit fails, do not proceed with the commit: fix the issue when clear, otherwise report it to
  the user.
